# 🛒 E-Commerce Web Application

A full-stack **E-Commerce Web Application** built using **Java, Spring Boot, MySQL, HTML, CSS, and JavaScript**. The application allows users to browse products, view product details, and manage product information through CRUD operations with MySQL database integration and REST APIs.

---

# 🚀 Tech Stack

| Category | Technologies |
|----------|--------------|
| 💻 Languages | Java |
| ⚙️ Frameworks | Spring Boot, Spring MVC  |
| 🎨 Frontend | HTML5, CSS3, JavaScript |
| 🗄️ Database | MySQL |
| 🛠️ Tools | IntelliJ IDEA,  Git, GitHub, Postman, MySQL Workbench |

# ⚙️ How to Run

1. Clone the repository
2. Create the MySQL database
3. Run `database.sql`
4. Update `application.properties`
5. Run `EcommerceApplication.java`
6. Open `http://localhost:8080`
